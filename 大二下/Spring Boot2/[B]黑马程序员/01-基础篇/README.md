# 01-基础篇

## 05R-基础篇总结
### 基于SpringBoot的SSMP整合案例
1. pom.xml
   - 配置起步依赖
2. application.yml
   - 设置数据源、端口、框架技术相关配置等
3. dao
   - 继承BaseMapper、设置@Mapper
4. dao测试类
5. service
   - 调用数据层接口或MyBatis-Plus提供的接口快速开发
6. service测试类
7. controller
   - 基于Restful开发, 使用Postman测试跑通功能
8. 页面
   - 放置在resources目录下的static目录中
