# 实验七 异常

### 【实验教学的目的】

1. 解错误和异常的基本概念及两者间的区别

2. 理解异常处理机制，掌握异常处理方法

3. 掌握抛出异常和捕捉异常的方法

### 【实验教学的基本要求】

认真完成实验内容，要求能够掌握异常处理的方法，能正确运行程序，判断实验结果的正确与否，并书写实验报告。

### 【实验内容】

1. 题目7-1：

   要求：用户输入的除数为0时，程序将发生错误。请修改程序，利用异常处理机制，解决程序中可能产生的异常，使程序正常运行

2. 题目7-2：

   要求：乘坐公交车时的刷卡操作。定义公交卡类，若刷卡金额大于金额，则作为异常处理。

### 【主要实验仪器及设备】

微型计算机、JDK、eclipse
