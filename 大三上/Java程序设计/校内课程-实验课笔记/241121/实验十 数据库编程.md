# 实验十 数据库编程（4学时）

### 【实验教学的目的】

1. 能在Java项目中加载JDBC驱动程序

2. 能用JDBC方式连接MySQL数据库

3. 掌握数据库的增删改查操作

### 【实验教学的基本要求】

认真完成实验内容，要求能够掌握数据库的连接方法和增删改查的操作，能正确运行程序，判断实验结果的正确与否，并书写实验报告。

### 【实验内容】

题目10-1：

编写一个Java程序，完成数据库表的创建，并使用JDBC分别完成数据的插入、删除、修改、查询操作。

### 【主要实验仪器及设备】

微型计算机、JDK、eclipse
