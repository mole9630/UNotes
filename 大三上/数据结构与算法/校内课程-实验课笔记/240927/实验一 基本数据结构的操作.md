# 实验一 基本数据结构的操作

### 实验目的

1. 掌握Dev C++ 的使用和调试方法。

2. 理解用户自定义数据类型。

3. 掌握用户自定义类型的用法。

### 实验预习

Dev C++的使用方法、C/C++的基本语法、用户自定义数据类型 

### 实验内容

具体内容：

1. 使用循环语句和选择语句，求1-100所有奇数的和。

2. 有5个学生，每个学生的数据包括学号、姓名、3门课程的成绩，从键盘输入5个学生数据，要求输出各学生3门课程平均成绩。
